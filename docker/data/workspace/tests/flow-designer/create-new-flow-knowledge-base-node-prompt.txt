## EDGE VERIFICATION RULE (MANDATORY — copy from create-new-flow-user-utterance-node.spec.ts)

Use count-based verification only. Before each drag, capture edge count:

  const edgesBefore = await page.locator('.vue-flow__edge[data-id]').count();
  // perform drag
  await page.waitForTimeout(1000);
  const edgesAfter = await page.locator('.vue-flow__edge[data-id]').count();
  if (edgesAfter <= edgesBefore) {
    throw new Error(`Edge NOT created — before: ${edgesBefore}, after: ${edgesAfter}`);
  }

NEVER use:
- [data-id^="e-START-"] or any source-based ID pattern (timestamp IDs — will never match)
- .vue-flow__edge.first() (matches ghost/preview edges)

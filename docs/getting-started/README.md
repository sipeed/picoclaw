## Prerequisites

Before you begin, you'll need:

- A computer running Linux, macOS, or Windows
- An API key from an LLM provider (OpenRouter, Zhipu, Anthropic, OpenAI, etc.)

## Quick Reference

```bash
# Install
picoclaw onboard

# Configure API key in ~/.picoclaw/config.json

# One-shot message
picoclaw agent -m "Hello!"

# Interactive mode
picoclaw agent

# Start gateway for chat apps
picoclaw gateway
```
